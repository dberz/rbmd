# Robin Berzin MD

Astro rebuild for `robinberzinmd.com`, deployed on Vercel.

## Local Development

```bash
npm install
npm run dev
```

The local dev server runs at `http://127.0.0.1:4321/`.

## Checks

```bash
npm run check
npm run build
```

## Environment Variables

Copy `.env.example` and set values in Vercel for production:

- `BEEHIIV_API_KEY`
- `BEEHIIV_PUBLICATION_ID`
- `INQUIRY_TO_EMAIL`
- `INQUIRY_FROM_EMAIL`
- `RESEND_API_KEY`
